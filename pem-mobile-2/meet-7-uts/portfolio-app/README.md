## Getting Started

cd pem-mobile-2/meet-7-uts/portfolio-app

### Prerequisites

- Node.js 16+
- Expo CLI: `npm install -g expo-cli`

### Installation

1. Clone the repository

2. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`

3. Start the development server:
   \`\`\`bash
   npm start
   \`\`\`

4. Open in Expo Go app or press:
   - `i` for iOS simulator
   - `a` for Android emulator
   - `w` for web
